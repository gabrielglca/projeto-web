# Sistema de Gestão Clínica

Este é um projeto acadêmico de um Sistema de Gestão Clínica desenvolvido em React com Vite. O objetivo do sistema é facilitar o controle de prontuários, consultas, médicos e receitas de forma prática.

## 🚀 Funcionalidades

- **Autenticação**: Controle de login administrativo simples.
- **Gestão de Pacientes**: Cadastro, edição e remoção de registros de pacientes.
- **Gestão de Médicos**: Cadastro e controle de informações médicas.
- **Agendamento de Consultas**: Registro e acompanhamento de consultas médicas.
- **Receituário**: Emissão e controle de receitas médicas.
- **Persistência Local**: Uso de `localStorage` para manter os dados no navegador.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: React.js
- **Ferramenta de Build**: Vite
- **Roteamento**: React Router DOM
- **Estilização**: CSS Vanilla
- **Banco de Dados**: LocalStorage (simulado no frontend)

## 📦 Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra o navegador no endereço indicado (geralmente `http://localhost:5173`).
